# Asset Index Template

## Conclusion

Use this file shape under `assets/index/` to make canonical assets discoverable.

## Rules

- Every canonical asset should appear in an index.
- Deprecated assets remain listed with status `deprecated`.
- Derived outputs should reference the source Asset ID.
