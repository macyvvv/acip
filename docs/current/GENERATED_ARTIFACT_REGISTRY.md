# GENERATED_ARTIFACT_REGISTRY

Generated artifacts are repository-managed outputs that validation and runtime tools may refresh explicitly.

## Registry

- `graph/agent_context_pack.json`
- `graph/repository_graph.json`
- `graph/repository_graph.md`
- `orchestrator/context_bundle.json`
- `orchestrator/execution_plan.json`
- `runtime/validation/validation_report.json`
- `runtime/validation/VALIDATION_REPORT.md`
- `runtime/generated_artifacts/generated_artifacts.json`
- `runtime/generated_artifacts/GENERATED_ARTIFACTS.md`

## Rule

- These paths are the canonical generated-artifact set for enforcement.
- Any dirty path outside this registry is treated as manual worktree change.
