# Audio Spec V2

## Audio Layers

1. Base
2. Character Noise
3. Action
4. Optional Voice

## Frequency Bands

- Mina: 200–800Hz
- Airi: 2k–6kHz
- Rena: ~1kHz pulses
- Nao: 8kHz+
- Yui: 4kHz–12kHz

## Noise Density

- Fixed: 0.3
- Yui: 0.2–0.3

## Trigger Points

- Before line (-0.3s)
- Gaze shift
- Silence entry
- Yui: before hesitation, before text line, after eye movement

## Silence Rule

- 0.8s after text is mandatory

## Sound Logo

- 0.2s minimal tone
- Must remain consistent across all videos

## Yui Profile

- Base layer: very low ambient tone, almost silence
- Noise type: soft hiss and intermittent micro crackle
- Silence rule: 0.8s after text is mandatory
- Voice: optional, but must remain hesitant and understated if used
