# DC-008: 記憶のガソリンスタンド / Gas Station of Memory

## コンセプト

砂漠の一本道にポツンと存在するガソリンスタンド。夕暮れが終わらない。給油機の数字がゼロのまま。自販機のすべてのボタンが「売切」。店内の時計は全部違う時刻を指している。ラジオからは自分の名前が呼ばれる天気予報が流れている。

## 映像構成

1. **導入（0:00-0:40）**: 地平線まで続く一本道。アスファルトの陽炎。遠くにガソリンスタンドの看板。「OPEN」のネオンが点滅。
2. **到着（0:40-2:00）**: ガソリンスタンドに到着。2台の給油機。どちらも数字表示が「¥0 / 0.00L」。キャノピーの蛍光灯が2本だけ点灯。地面にオイル染み。自販機に近づく。すべて「売切」。コイン投入口に硬貨を入れると、そのまま返却口に落ちてくる。
3. **店内（2:00-3:30）**: ガラスドアを開けて店内へ。カウンターに誰もいない。壁掛け時計が3つ。12:00、3:15、永遠に回り続ける秒針だけの時計。棚には商品があるが、すべてのラベルが白紙。冷蔵庫のドリンクは中身が透明。
4. **放送（3:30-4:30）**: カウンターのラジオが鳴り出す。天気予報。「明日の◯◯（自分の名前）地方は…」。振り返ると窓の外の夕陽が全く動いていない。外に出ると影の方向が来た時と同じ。
5. **終結（4:30-5:00）**: 一本道の先を見る。同じガソリンスタンドがもう一つ見える。その先にも。無限に連なるガソリンスタンド。

## 音響設計

- **基層**: 砂漠の風（乾いた低いヒュー音）、遠くのハイウェイの走行音
- **中層**: 蛍光灯のジー音、自販機のコンプレッサー音
- **上層**: ラジオの天気予報（AMラジオの質感）
- **終結**: 風の音だけが残り、他のすべてが消える

## 感情設計

乾燥した孤独 → 微かな希望 → 不条理の受容 → 名前を呼ばれる衝撃 → 永遠の反復

## 想定尺

5分

## 制作上の要点

- アメリカ的なガソリンスタンドではなく、日本のロードサイド（ENEOS/出光的な）をベースに砂漠に置く異化効果
- 夕暮れの色はゴールデンアワー（太陽高度5°）で固定
- 「自分の名前」は実際の音声では伏せ字的に処理（ビープ音で隠す）

- **0:00 - 0:15**: 視覚的フック。映像の冒頭から最も奇妙で美しいリミナルスペースの光景を提示。足音などの環境音のみで静寂を強調し、視聴者を引き込む。
- **0:15 - 0:45**: メカニズムの提示。説明的ナレーションは一切なし。カメラはゆっくりと前進または回転し、空間の歪みや「もう一人の自分」などの異変を客観的に映し出す（標本的扱い）。
- **0:45 - 1:00**: 無限ループ設計。映像のラストシーンが、冒頭の最初のカットと視覚的・音響的に完全に一致し、ループ再生した際に境界がわからないようにシームレスに接続して終了。
