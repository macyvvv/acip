# DreamCore Video

DreamCore Video は ACIP の動画コンテンツ事業。

## Purpose

DreamCore系（リミナルスペース・超現実的映像）の高品質動画コンテンツを制作・配信する。

## Strategy

- **市場**: レッドオーシャン（先駆者多数）
- **判断根拠**: 普遍的に受け入れられるジャンルであり、いくらあっても需要がある
- **差別化**: 映像品質、音響設計、感情の弧の三層で統一された世界観を構築

## Structure

- `ideas/`: 動画台本アイデア（Core: DC-001 ~ DC-012, Loop-First: DCL-001 ~ DCL-012）
	- Core Pack: DC-001 ~ DC-012
	- Loop-First Pack: DCL-001 ~ DCL-012

## Operating Principles

- 映像美と不安感のバランスを維持する
- 既視感と非日常の境界を設計対象とする
- 音響は映像と同等の設計要素として扱う
- 各動画は独立した世界として完結させる

## Direction Update (2026-07-15)

ストーリー性は活かすが、主軸は「説明されない違和感の持続」に置く。

### Creative Intent

- 物語の起承転結よりも、終わらない感覚を優先する
- 恐怖は具体化しすぎない（正体・原因・犯人を示さない）
- 視聴者が気づかないうちにループ再生される体験を設計する

### Loop-First Rules

- ループ境界を消す: ラスト1-2秒の構図・動き・環境音を冒頭と一致させる
- 時間の固定化: 時計、信号、エレベーター階数などが進まない
- 空間の自己反復: 歩いても同じ地点へ戻る、降下し続けるのに到達しない
- 因果の欠落: 変化は起きるが理由は示さない

### Fear Design (Abstract)

- 明確な敵・怪物・説明台詞に頼らない
- 断片だけを見せる（影、反響、温度感、欠けた情報）
- 違和感は「微差の反復」で増幅する（毎ループで1要素だけ僅かに変える）

### Retention Design

- 8-15秒ごとに小さな異変を配置し、離脱を防ぐ
- 音響の低域ドローンを一定に保ちつつ、周辺ノイズだけ更新する
- 終了感を作らない（解決・種明かし・明確なエンディングを避ける）

### Prohibited by Default

- ジャンプスケアの多用
- グロテスク描写
- 恐怖要因の明示的説明
