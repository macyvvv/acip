# 身近な機械の意外なカラクリ

身近な道具や機械の中で起きている仕組みを、ショート動画向けに分かりやすく図解するシリーズ。

## コンセプト

- 対象: ボールペン、シャープペン、錠前、ポンプ、車、家庭機器など
- 価値: 毎日使っているのに知らない「なぜ動くのか」を60秒で理解できる
- 体験: 見終わった直後に実物を手に取りたくなる

## シリーズ方針

- 冒頭3秒で疑問を提示する
- 断面図、分解図、矢印アニメで力と動きを可視化する
- 難しい用語は最小限にし、必ず身近な例に戻す
- 1動画1メカニズムに絞る

## 出力フォーマット

- プラットフォーム: TikTok / YouTube Shorts / Reels
- 尺: 45-60秒
- 画面: 9:16 縦型
- 音: BGM + 効果音 + テロップ中心

## ファイル構成

- ideas/MECH_SHORTS_100_IDEAS.md: 台本アイディア100本
- SERIES_CONCEPT.md: 企画の目的、対象視聴者、差別化
- SHORTS_SCRIPT_TEMPLATE.md: 台本テンプレート
