# PMV-1103: ガンマ関数と階乗の複素平面：滑らかにつながる離散世界 / The Gamma Function

## 核心概念
自然数 $n$ に対する階乗 $n!$ を、複素数全体（負の整数を除く）へと滑らかに拡張するガンマ関数 $\Gamma(s) = \int_0^\infty x^{s-1}e^{-x} dx$ の幾何学。

## 既存コンテンツの問題点
- 積分の計算公式や部分積分による $\Gamma(s+1) = s\Gamma(s)$ の導出ばかりで、なぜこの積分によって「飛び飛びの階乗の点」が完全に滑らかな美しい谷（曲面）で接続されるのかの視覚的プロセスがない。

## ビジュアライゼーション手法
1. 2次元グラフ上に飛び飛びに並んだ階乗の点 $(1,1), (2,2), (3,6), (4,24)...$ を配置。
2. ガンマ関数の積分が、実数軸上でこれらの点を結ぶ滑らかな曲線を「自動的に補間」して描き出すプロセスを可視化。
3. 複素平面全体に拡張された $\Gamma(s)$ の3D絶対値曲面を描画。負の整数 $s = 0, -1, -2...$ において、関数が無限大に吹き飛ぶ「無数の特異点（ポール）」が規則正しく林立する幾何的ランドスケープを示す。

## 核心的洞察
ガンマ関数は、離散的な世界（階乗）を連続的かつ正則な世界へと翻訳する。負の整数で無限大になる特異点の配置こそが、この関数の滑らかな形状を決定づける骨組みである。

## 前提知識
階乗、定積分
