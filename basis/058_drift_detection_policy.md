# 058 Drift Detection Policy

## Conclusion

Repository drift occurs when files, policies, ADRs, WBS, or workflows contradict the active Operating Model.

## Drift Types

- Current Objective drift
- Human Boundary drift
- Runtime Boundary drift
- terminology drift
- duplicate governance drift
- stale validation drift
- workflow-script drift

## Rule

Drift must be reported with file path, evidence, and recommended owner.
