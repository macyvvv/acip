# 026 Autonomy First Policy

## Conclusion

ACIP prioritizes autonomous execution paths so Human remains focused on Mission, Approval, and Emergency Stop.

## Human Boundary

Human handles:

- Mission
- Approval
- Emergency Stop
- Risk Acceptance
- Capital Allocation
- Runtime Transition Approval

## Delegation Rule

If a task can be safely performed by ChatGPT, Codex, validation scripts, GitHub Actions, or approved future automation, it should not be assigned to Human by default.

## Current Boundary

Runtime implementation remains out of scope until explicitly approved.

## Repository Rule

Repository overrides conversation.
