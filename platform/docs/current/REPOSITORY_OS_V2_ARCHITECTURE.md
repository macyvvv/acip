# REPOSITORY_OS_V2_ARCHITECTURE

## Architecture Overview

Repository OS v2 is a layered, projection-driven operating baseline.

### Layers

- Constitution
- Planning State
- Repository State
- Supervisor
- Work Planner
- Validation

### Operating Principle

Authoritative SSOTs remain authoritative. Projections aggregate them without owning independent state.

### Release Position

This architecture is frozen as the baseline for subsequent product development.
