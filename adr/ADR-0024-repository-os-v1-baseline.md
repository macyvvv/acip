# ADR-0024: Repository OS v1.0 Baseline

## Status

Proposed

## Context

ACIP has completed Repository OS, Knowledge Factory, Semantic SelfTest v2, Repository Knowledge Graph, Agent Context Pack, Agent Orchestrator, and Runtime Transition Readiness.

## Decision

Adopt `v1.0.0-repository-os` as the baseline for future runtime and agent development.

## Consequences

- Future changes can be compared against a stable baseline.
- Runtime work requires explicit transition approval.
- Validation and graph-diff workflows become meaningful.
