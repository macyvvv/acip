# RB-0005: Validation Failure Recovery

## Objective

Resolve validation failure without unnecessary Human involvement.

## Steps

1. Identify failed check.
2. Map failure to required file or required text.
3. Determine whether fix is mechanical.
4. Assign mechanical fix to Codex.
5. Escalate only when requirement is ambiguous or approval is required.
6. Re-run validation.
7. Report final status.

## Human Boundary

Human is not asked to debug validation failures unless risk acceptance or policy change is required.

## Done Condition

Validation passes or escalation record exists.
