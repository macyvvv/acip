# Purpose

This directory stores Canonical Knowledge Assets.

A Canonical Asset is a reusable, long-lived knowledge document that improves Human QOL through AI Native Execution.

This directory is a production repository.

It is NOT a memo directory.

It is NOT a draft directory.

---

# Mission

Build reusable knowledge.

Every document should continue to create value months and years after creation.
