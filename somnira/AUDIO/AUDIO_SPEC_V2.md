# Audio Spec V2

## Audio Layers

1. Base
2. Character Noise
3. Action
4. Optional Voice

## Frequency Bands

- Mina: 200–800Hz
- Airi: 2k–6kHz
- Rena: ~1kHz pulses
- Nao: 8kHz+

## Noise Density

- Fixed: 0.3

## Trigger Points

- Before line (-0.3s)
- Gaze shift
- Silence entry

## Silence Rule

- 0.8s after text is mandatory

## Sound Logo

- 0.2s minimal tone
- Must remain consistent across all videos
