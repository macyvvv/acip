# Somia

Somia is ACIP’s AI Character Media business unit.

## Purpose

Define a reproducible IP generation system for character-driven media assets.

## SSOT

- GitHub repository artifacts are canonical.
- Specs are written before implementation.
- Content production must stay deterministic and reviewable.

## Structure

- `BRAND/`: brand identity and logo rules
- `CHARACTER/`: character-level specifications
- `VISUAL/`: visual identity rules
- `VIDEO/`: short-form video formatting rules
- `AUDIO/`: audio layer and timing rules
- `PRODUCTION/`: pipeline and asset organization
- `CONTENT/`: canonical content outputs

## Operating Principles

- Separate brand, character, visual, audio, and pipeline concerns.
- Keep unresolved relationships intentional.
- Maintain ambiguity where it is part of the concept.
- Preserve reproducibility across future content runs.
