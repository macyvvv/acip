# Visual Identity

## Abstract Background Rules

- Backgrounds should feel psychological, not realistic.
- Space should support unresolved presence.
- Avoid clutter that competes with the character silhouette.

## Accessory Rule

- Accessories are fixed identity markers.
- The accessory should remain recognizable across outputs.
- Do not swap accessories casually between assets.

## Character Color Palette

- Mina: muted violet
- Airi: cool blue
- Rena: deep red
- Nao: graphite
- Yui: pale pink, soft ivory, faint gray shadow

## Character Visual Notes

- Yui’s fixed accessory is a small stuffed rabbit.
- Yui uses black twin tails with slightly uneven symmetry.
- Yui’s background should feel like a soft room with blurred edges and unstable space.

## Lighting Rules

- Keep lighting soft but intentional
- Use contrast to support ambiguity
- Avoid documentary-style realism
- Yui should use soft diffused light with slightly overexposed highlights and weak shadows

## Noise / Texture Rules

- Texture should be restrained
- Add noise only if it supports mood
- Never use texture to hide composition problems
- Yui may use fine grain and slight bloom to support fragility
